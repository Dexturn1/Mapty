# 🗺️ Mapty

Mapty is a browser-based workout tracking application that allows users to log and visualize their running and cycling activities on an interactive map.

---

## 🚀 Features

- 📍 Click on the map to add a workout
- 🏃 Track running workouts (distance, duration, cadence)
- 🚴 Track cycling workouts (distance, duration, elevation gain)
- 🗂️ View workouts in a list
- 🔍 Automatically zoom to workout location
- 💾 Persistent data using local storage

---

## 🧠 Concepts Used

This project demonstrates:

- Object-Oriented Programming (OOP) in JavaScript

## ⚙️ How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/Dexturn1/Mapty.git
   ```

2. Navigate into the project:

   ```bash
   cd Mapty
   ```

3. Open `index.html` in your browser
   (or use Live Server in VS Code)

---

## ⚠️ Important Notes

- Geolocation must be enabled in your browser
- Works best on modern browsers (Chrome, Edge, Firefox)
